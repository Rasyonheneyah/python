# 🐍 Python Exercises - Exercícios de Programação

Repositório com exercícios de **Python** organizados por temas e níveis de dificuldade.  
Criado com o objetivo de consolidar fundamentos da programação, desenvolver raciocínio lógico e construir uma base sólida para projetos maiores.

---

## 📂 Estrutura do Repositório

### 📁 basicos/
Variáveis, tipos de dados, entrada e saída, operadores e primeiros programas.

### 📁 condicionais/
Estruturas de decisão (`if`, `elif`, `else`) e lógica booleana.

### 📁 loops/
Estruturas de repetição (`for`, `while`), controle de fluxo e padrões com repetição.

### 📁 funcoes/
Criação de funções, parâmetros, retorno de valores e organização modular do código.

### 📁 estruturasDeDados/
Listas, tuplas, dicionários e conjuntos. Manipulação, iteração e aplicações práticas.

### 📁 manipulacaoDeStrings/
Fatiamento, métodos de string, formatação e validação de textos.

### 📁 tratamentoDeErros/
Uso de `try`, `except`, `else` e `finally` para tornar os programas mais seguros e robustos.

### 📁 desafios/
Problemas que exigem raciocínio lógico mais elaborado, combinando múltiplos conceitos.

### 📁 praticasAvulsas/
Exercícios em andamento, testes e experimentações.

---

## 🎯 Objetivo

- Praticar lógica de programação  
- Consolidar conceitos fundamentais  
- Evoluir progressivamente na linguagem  
- Criar base sólida para projetos maiores
